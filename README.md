# PG Drive - Rental Management Platform

PG Drive is a full-stack rental management platform that helps users discover and manage PG accommodations and bike rental services. The platform provides separate modules for Users and Owners with secure authentication, booking management, and responsive user experience.

## Features

- User Registration & Login
- Owner Registration & Login
- Role-Based Authentication
- PG Listing & Search
- Bike Rental Management
- Booking Management
- Responsive User Interface
- REST API Integration
- Secure Authentication
- Mobile-Friendly Design

## Tech Stack

### Frontend

- React.js
- JavaScript (ES6+)
- React Router
- Bootstrap
- CSS3
- Axios

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Installation

### Clone Repository

```bash
git clone https://github.com/ritasahaa/pg-drive-client.git
```

### Install Dependencies

```bash
npm install
```

### Run Application

```bash
npm start
```
